##  What is SPECKit Plus?
**SPECKit Plus** is a structured framework to make software development **simpler and organized**.  
It guides students **step-by-step** from defining an idea to delivering a final working solution.  
The framework has **five core phases**: Constitution, Specify,Plan,Tasks, Implement.
